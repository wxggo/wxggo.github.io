Java学习笔记：子数组和等于k的个数求解

Created2024-10-08|Updated2024-10-08|Java学习笔记
|Post Views:

1、题目

给定一个整数数组 nums 和一个整数 k ，请你返回该数组中和为 k 的连续子数组的个数。

示例

示例 1：
输入：nums = [1,1,1], k = 2
输出：2

示例 2：
输入：nums = [1,2,3], k = 3
输出：2

提示

1 <= nums.length <= 2 * 10⁴
-1000 <= nums[i] <= 1000
-10⁷ <= k <= 10⁷

2、分析

暴力解法的局限

首先思考最直观的暴力解法：遍历所有可能的子数组，计算其和并统计等于k的个数。这种思路需要两层循环（外层确定子数组起点，内层确定终点），时间复杂度为O(n²)。但题目中数组长度最大为210⁴，O(n²)的时间复杂度会导致运算量达到410⁸，容易超时，因此需要更高效的解法。

前缀和+哈希表优化思路

前缀和定义

我们定义前缀和数组s，其中s[0] = 0，s[i] 表示原数组nums中前i个元素的和（即nums[0] + nums[1] + … + nums[i-1]）。根据定义可得：
sum(nums[j...i-1]) = s[i] - s[j]

核心逻辑

题目要求子数组和等于k，即：
s[i] - s[j] = k
变形后可得：
s[j] = s[i] - k

这意味着，对于当前遍历到的前缀和s[i]，我们只需要统计之前出现过的、值为s[i]-k的前缀和s[j]的个数，这个个数就是以i-1为终点、和为k的子数组的数量。

为了高效统计前缀和出现的次数，我们使用哈希表（HashMap） 存储「前缀和值 → 该值出现的次数」。遍历前缀和数组时，先查询哈希表中是否存在s[i]-k，若存在则将对应次数累加到答案；再将当前前缀和s[i]加入哈希表（若已存在则次数+1，否则设为1）。

算法复杂度

  • 时间复杂度：O(n)，仅需遍历原数组计算前缀和，再遍历前缀和数组一次，哈希表的查询/插入操作均为O(1)。
  • 空间复杂度：O(n)，前缀和数组和哈希表的空间开销均为O(n)。

3、程序

完整代码

代码逐行解析

  1. int n = nums.length;
    获取输入数组nums的长度，后续用于构建前缀和数组，确定遍历边界。

  2. int[] s = new int[n + 1];
    构建前缀和数组s，长度为n+1（s[0]作为初始值0，s[1]对应nums[0]，s[2]对应nums[0]+nums[1]，以此类推），避免处理子数组起点为0时的边界越界问题。

  3. for (int i = 0; i < n; i++) { s[i + 1] = s[i] + nums[i]; }
    遍历原数组计算前缀和：

    • i=0时，s[1] = s[0] + nums[0] = 0 + nums[0]（前1个元素的和）；
    • i=1时，s[2] = s[1] + nums[1] = nums[0] + nums[1]（前2个元素的和）；
    • 以此类推，最终s数组完整存储从s[0]到s[n]的所有前缀和。

  4. int ans = 0;
    初始化答案变量，用于累计所有和为k的连续子数组个数。

  5. Map<Integer, Integer> cnt = new HashMap<>(n + 1);
    创建哈希表cnt，键为「前缀和值」，值为「该前缀和出现的次数」；初始容量设为n+1，减少哈希表扩容次数，提升执行效率。

  6. for (int sj : s) { ... }
    遍历前缀和数组中的每一个值sj（覆盖s[0]到s[n]），核心逻辑分为两步：

    • ans += cnt.getOrDefault(sj - k, 0);
      查询哈希表中是否存在键sj - k（即满足s[j] = s[i] -k的前缀和）。getOrDefault方法保证：存在则返回对应次数，不存在则返回0，将结果累加到ans中，这一步直接统计出以当前位置为终点的、和为k的子数组数量。
    • cnt.merge(sj, 1, Integer::sum);
      将当前前缀和sj存入哈希表：若sj未存在，直接存入键值对(sj, 1)；若已存在，将对应值+1（Integer::sum表示旧值与新值1相加）。

  7. return ans;
    遍历完成后，ans即为所有满足条件的子数组个数，返回结果。

示例推演（以nums=[1,1,1], k=2为例）

  1. 计算前缀和数组s：

    • s[0] = 0
    • s[1] = 0 + 1 = 1
    • s[2] = 1 + 1 = 2
    • s[3] = 2 + 1 = 3

  2. 遍历前缀和数组s（哈希表cnt初始为空）：

    • 遍历sj=s[0]=0：
      ans += cnt.getOrDefault(0-2=-2, 0) → ans=0
      cnt.merge(0,1,…) → cnt={0:1}
    • 遍历sj=s[1]=1：
      ans += cnt.getOrDefault(1-2=-1, 0) → ans=0
      cnt.merge(1,1,…) → cnt={0:1, 1:1}
    • 遍历sj=s[2]=2：
      ans += cnt.getOrDefault(2-2=0, 0) → ans=1
      cnt.merge(2,1,…) → cnt={0:1, 1:1, 2:1}
    • 遍历sj=s[3]=3：
      ans += cnt.getOrDefault(3-2=1, 0) → ans=2
      cnt.merge(3,1,…) → cnt={0:1, 1:1, 2:1, 3:1}

  3. 返回ans=2，与示例结果一致，验证算法正确性。
