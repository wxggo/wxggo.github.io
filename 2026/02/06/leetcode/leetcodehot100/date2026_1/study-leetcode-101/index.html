Leetcode-101:对称二叉树-递归法极简实现

Created2026-02-06|Updated2026-02-06|Java学习笔记Leetcode Hot100
|Word Count:1.6k|Reading Time:5mins|Post Views:

1、题目

给定一个二叉树，检查它是否是镜像对称的。要求判断二叉树的左右子树是否互为镜像，即结构对称且对应节点值相等。

示例

示例 1：

输入：root = [1,2,2,3,4,4,3] 输出：true

示例 2：
输入：root = [1,2,2,null,3,null,3]
输出：false

提示

  • 树中节点数目范围在 [1, 1000] 内
  • -100 <= Node.val <= 100

2、分析

2.1 问题核心

判断二叉树是否对称的核心是验证左右子树是否互为镜像，关键难点是处理节点为空的边界情况（如单侧子树为空、双侧子树都为空），以及递归过程中“镜像位置”的节点匹配（左子树的左节点对应右子树的右节点，左子树的右节点对应右子树的左节点）。

递归法是该题的最优解之一：通过递归函数逐层判断镜像位置的节点是否满足对称条件，逻辑直观且代码简洁，无需额外数据结构辅助。

2.2 核心思路

  1. 边界处理：若根节点为空，直接返回true（空树天然对称）；
  2. 递归函数定义：设计私有递归函数，接收两个节点（左子树节点left、右子树节点right），判断这两个节点所在的子树是否互为镜像；
  3. 递归终止条件：
    • 若left和right都为空，说明镜像位置无节点，返回true；
    • 若left和right仅有一个为空，说明结构不对称，返回false；
  4. 递归核心逻辑：
    • 首先判断当前两个节点的值是否相等；
    • 递归判断left的左子节点与right的右子节点是否对称；
    • 递归判断left的右子节点与right的左子节点是否对称；
    • 三者都满足时，才返回true；
  5. 触发递归：从根节点的左右子节点开始调用递归函数，返回最终结果。

3.2 代码简洁解释

  • 根节点判空 ：空树直接返回true，处理最基础的边界场景；
  • 递归函数isSymmetric(TreeNode left, TreeNode right) ：核心判断逻辑的封装，专注于两个镜像位置节点的校验；
  • 终止条件left == null || right == null ：通过left == right快速判断：双空则对称（true），一空一非空则不对称（false）；
  • 递归核心表达式 ：
    • left.val == right.val：保证当前镜像节点值相等；
    • isSymmetric(left.left, right.right)：保证左子树的左节点与右子树的右节点对称；
    • isSymmetric(left.right, right.left)：保证左子树的右节点与右子树的左节点对称；
    • 三者需同时满足，才能说明子树镜像对称。

4、算法性能

  • 时间复杂度：O(n)（n为二叉树的节点数）。每个节点仅被递归遍历一次，无重复计算；
  • 空间复杂度：O(n)。递归调用的栈深度取决于二叉树的高度，最坏情况下（二叉树退化为链表），栈深度等于节点数；最优情况下（平衡二叉树），栈深度为logn；
  • 核心优势：
    1. 递归实现逻辑直观，代码量极少，易理解易书写；
    2. 无需额外数据结构（如队列/栈），空间开销仅为递归栈；
    3. 天然适配二叉树的递归结构，符合“分治”解题思路。

5、测试示例

以root = [1,2,2,3,4,4,3]为例，执行过程如下：

  1. 根节点1非空，调用isSymmetric(2, 2)；
  2. left=2、right=2都非空，值相等，继续递归：
    • 调用isSymmetric(3, 3)（左2的左节点 vs 右2的右节点）；
    • 调用isSymmetric(4, 4)（左2的右节点 vs 右2的左节点）；
  3. 处理isSymmetric(3, 3)：
    • 3的左右节点都为空，返回true；
  4. 处理isSymmetric(4, 4)：
    • 4的左右节点都为空，返回true；
  5. 所有递归条件满足，最终返回true，与示例一致。

以root = [1,2,2,null,3,null,3]为例：

  1. 根节点1非空，调用isSymmetric(2, 2)；
  2. left=2、right=2值相等，递归调用isSymmetric(null, 3)（左2的左节点 vs 右2的右节点）；
  3. null和3一空一非空，返回false，最终整体返回false，与示例一致。

6.2 关键边界情况处理

代码天然适配所有边界场景，无需额外判断：

  • 空树：根节点判空直接返回true；
  • 单节点树：根节点非空，左右子节点都为空，递归返回true；
  • 单侧子树为空：递归终止条件直接识别，返回false；
  • 节点值不相等：递归核心逻辑第一步就会返回false。

6.3 拓展应用

该递归思路可扩展至判断两棵二叉树是否互为镜像（Leetcode剑指Offer28），只需将递归函数的入参从“根节点的左右子树”改为“两棵树的根节点”，核心判断逻辑完全复用。
